Common Questions

Is there really no JavaScript?

Correct. Everything you see, including the mobile menu drawer, the sticky header, and the form interactions, is built using pure CSS3.

How does the menu work?

It uses the "Checkbox Hack". We have a hidden checkbox input. When you click the label (the menu button), it checks the box. CSS selects the adjacent navigation menu and changes its 'right' position to slide it in.

What about the text boxes?

These use the :focus and :valid pseudo-classes to detect when a user clicks inside or types text, moving the label upwards.
